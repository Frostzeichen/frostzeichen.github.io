Button Counter


The Button Counter will count how many times you press the red button.


Counter: 0


Explanation:

The button counter app uses three files: Easy_Sample_1.html, Easy_Sample_1.css, and Easy_Sample_1.js.

In order to make it run, I made the JavaScript file wait for a click from the red button. Whenever a click is made, the JavaScript file would update the number on the counter by one.

The reset counter brings the count back to zero.
